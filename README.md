### 📄 Research

**A Lightweight MobileNetV4 Framework for Efficient Multi-Class Brain Tumor MRI Classification**
*1st Author — Accepted at IEEE TENCON 2026, Bali (Oct 10–13)*
Achieved **96.91% test accuracy** and **97.20% F1-score** on multi-class medical image classification.

**SmartSnack: RFID-Based Tangible User Interface for Dietary Safety Advisory**
*1st Author — Submitted to ICCA 2026 (HCI & User-Centered Systems)*
A physical interface helping users with Type 2 diabetes and gluten intolerance evaluate food safety in **under 0.6 seconds per action**, with usability testing showing a significant reduction in cognitive load vs. text-based dietary apps.
